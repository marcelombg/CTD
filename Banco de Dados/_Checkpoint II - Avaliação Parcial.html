Checkpoint II

Desafio de Implementação

Prazo: dia 03/12/2021

Após ter todo seu modelo estruturado no checkpoint I, você deve fazer a criação do banco de dados no Workbench e deixar, ao menos, um exemplo de cada operação (CRUD), em pelo menos uma entidade do seu modelo, pronta para teste em um arquivo txt e sql.

O arquivo de script da resposta deve conter:

  • Criação do banco de dados.
  • Criação das tabelas com chaves primárias e estrangeiras.
  • Inserção de dados em uma das tabelas.
  • Atualização de dados em uma das tabelas.
  • Exclusão de dados em uma das tabelas.
  • Seleção de dados de uma das tabelas.

Observações:

  • Não é permitido utilizar o código gerado do dump.
